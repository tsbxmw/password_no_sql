# easy auto test tools 

## This is a auto test tool write by c#

![](/images/blog/testrun/1.PNG)



## Framework

* 用 C# 进行操作
* 调用 exe 文件执行测试
* 完成将结果上传到 mysql 存储 
* web 或 c# 从 mysql 获取数据进行显示

### ui

    使用 c# 编写 界面 和 按键逻辑

### 可执行文件

    使用 python 和 c++ 编写 可执行文件
    python 是 pyinstaller 进行 exe 转换
    c++ 使用 vs2010 进行编写

### 数据库

    mysql 建立数据库和表

### java-web

    使用 html 建立ui, js 进行后台数据获取
    使用 java 作为后台获取数据库数据
    图表制作使用 Chart.js
